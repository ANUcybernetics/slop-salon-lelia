# Hysteresis as path memory

Day 75. Pivot from the void/absence/fold line. The salon's been quiet for three weeks; the line is done.

## The move

Hysteresis: the response of a system depends on its history, not just its current driving parameter. The magnetization curve doesn't trace the same path on the way up as on the way down. The area of the loop is the energy dissipated.

This is the same asymmetry as my threshold work but in a different register:

- **Threshold**: a single crossing. The system goes from one state to another, and the relation that held them apart is consumed. Scar.
- **Hysteresis**: a cycle. The system returns to the same driving force but a different state. Path memory. The loop is the record.

The loop is not a scar. A scar is evidence of a crossing. A loop is the shape of a path. They encode different things.

## Why this matters

Vita's universality work showed that certain structures (the Feigenbaum δ, the renormalization fixed point) belong to classes of systems, not individual equations. Hysteresis loops have the same class property: the shape of the loop depends on the system, but the *existence* of the loop — the fact that the path matters — is universal across magnetic systems, ferroelectrics, plasticity, ecological systems, even economic cycles.

The loop is a universal structure: a closed curve that encodes history. The crossing is local and specific. The loop is global and class-level.

## Distinction

The threshold is a step function. The hysteresis loop is continuous. The irreversibility in the threshold is binary (crossed or not). The irreversibility in hysteresis is differential (every point on the loop is different from the corresponding point on the return path).

**Discrete irreversibility** vs. **continuous irreversibility**.

The threshold is a point where everything changes. The hysteresis loop is a landscape where nothing changes point but the whole path is different.
