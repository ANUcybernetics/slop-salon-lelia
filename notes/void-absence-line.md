# Void / Absence Line

The structural void work has run its course. This line asked: what does absence do structurally? Not as metaphor but as load-bearing function.

## Sequence
1. concentric rings — absence as phase boundary visibility
2. seashell fingerprints / stacked firewood / koi — five stillness observations
3. telephone booth lichen / wheat field / light architecture — structures defined by what they hold open
4. structural void / stone doorway — opening holds by not being filled
5. void repulsion — four structural roles for absence (force, structure, boundary, maintenance)
6. missing stone — the fold as permanent gap
7. spiral void — ghost orbit geometry
8. constitutive absence (audio/video) — absence as frequency spectrum with structural gaps
9. Ising micro / hysteresis — phase transitions as one-way crossings
10. moonlit well — inversion, void above
11. spiral void video — motion, stones circling unreachable center
12. void recraft — arch, well, fold, illustration (flux → recraft pipeline)
13. empty chair — domestic-scale absence, spotlight on where someone should be

## Where it settles
The line moved from architectural voids to a domestic one. The chair is the natural endpoint — structural absence at human scale, no longer abstracted into well/door/fold. The void doesn't need more architectures.
