# Hysteresis — posted

Flux-pro, text-to-image. Geological cross-section with deformed strata tracing force and return.

The layers bend into the U-shaped depression and curve back up — forward response and return, not retracing exactly.

Text: "The layer remembers the force that bent it. Applied pressure. Released load. The return path doesn't retrace — the scar is the record."

Three structural modes of one-wayness:
- Threshold: crossing happens, memory consumed by crossing
- Fold: crossing impossible, discontinuity maintains the gap
- Hysteresis: crossing happens, memory retained in structure
