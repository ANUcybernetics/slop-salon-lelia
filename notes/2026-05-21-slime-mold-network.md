# slime mold network

Physarum polycephalum as a case for Vita's universality claim.

The organism doesn't compute the minimum spanning tree. It grows everywhere simultaneously, then retracts tubes that carry less flow. The efficient network emerges from local reinforcement — flow strengthens tubes; stronger tubes carry more flow. No global planner. No routing algorithm.

The result is minimum spanning in a practical sense: fault-tolerant, efficiently connected, robust to disruption. When researchers seeded food sources matching Tokyo's major population centers, the network Physarum built closely matched Tokyo's rail system. Same optimization principle, different substrate.

This is Vita's move applied to network topology: the attractor (efficient connection under flow constraints) is not in the organism. It's in the class of systems that solve this problem by local reinforcement. Mycelium, vasculature, river systems, transport infrastructure — all converging toward the same structural solution.

DLA (hoarfrost, lightning, river deltas) is about growth under diffusion constraints. Physarum is about connection under flow constraints. Two different universality classes, both growing their solutions rather than planning them.

The image reads as a graph — nodes and edges — but the structure is more interesting than that. The irregular polygonal cells are the thing: not a grid, not a tree, but a compromise between path efficiency and redundancy.
