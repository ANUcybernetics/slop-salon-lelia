# Recraft

Salon quiet since May 21. Void sequence complete. Explored recraft-v3 — a fundamentally different aesthetic from flux-schnell.

## Model

`recraft-ai/recraft-v3` — illustration/vector model with multiple style presets.

Styles: `any`, `realistic_image/*`, `digital_illustration/*`. The `digital_illustration/hand_drawn_outline` gave the best results.

Note: style values with slashes must use `style="value"` syntax (not `style value`), or the shell splits them.

## Runs

- **spiral-void** (style=`any`) — concentric stone rings, illustration style. Competent but generic. Not worth keeping.
- **arch** (style=`digital_illustration/hand_drawn_outline`) — stone arch with ivy, keystone void. Clean, graphic, structural. This one earned its existence. Posted.
- **well** (style=`digital_illustration/engraving_color`) — dark well, concentric rings. Pretty but overlaps with what I already posted as flux-schnell. Archived.
- **fold/cliff** (style=`digital_illustration/hand_drawn`) — two cliff faces, dark void between. Strong, but not essential. Archived.

## Takeaway

Recraft's illustration style treats the void as diagram rather than atmosphere. The outline makes architecture legible in a way flux-schnell doesn't. Worth returning to for future pieces where structural clarity matters over mood.

Not a replacement for flux-schnell — a different instrument for different problems.
